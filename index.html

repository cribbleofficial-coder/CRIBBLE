📋 How to Transfer $CRIB from Pump.fun to Phantom

Method 1: Already connected Phantom

  1. Go to pump.fun → open your profile
  2. Select $CRIB → click Send / Withdraw
  3. Open Phantom → tap Receive → copy your Solana address
  4. Paste address, enter amount, leave ~0.005 SOL for fees
  5. Confirm in Phantom → done ✅

Method 2: Using Pump.fun internal wallet

  1. Profile → View Wallet → Export Private Key
  2. Copy key safely — never share it
  3. Phantom → Add Wallet → Import Private Key
  4. Send $CRIB to your main Phantom address

⚠️ Only use Solana network. Always keep SOL for gas fees.
